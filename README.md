# Ghost Character Typesetter (鬼畜活字乱刷)

A video character-level editing tool with AI speech recognition, built on Electron.

Download the latest installer from [Releases](https://github.com/lemonungood/guichu-tool/releases).

## Usage

1. Install and launch the app
2. Create a project → import a video
3. AI recognizes speech and builds a character library with timestamps
4. Arrange characters on the timeline
5. Export the final video

Supports plugins, UI themes, and multi-project management.
