# 插件系统

> 📖 完整插件开发指南请查看：[`docs/plugin-guide.md`](docs/plugin-guide.md)

## 快速开始

1. 编写符合规范的插件，打包为 `.zip` 文件
2. 在应用内打开 **插件管理器** → **导入插件(.zip)**
3. 系统自动解压、检测兼容性、加载

## 核心概念

- 📦 **分发格式**: `.zip` 压缩包，内含 manifest.json + JS代码 + 素材
- 🔄 **自恢复**: 插件目录被删除后应用启动时自动重建
- ⚡ **优先级**: 插件 `priority` 数值越高越优先，插件行为 > 核心行为
- 🚫 **不兼容检测**: 同名字段 + `incompatible` 声明双重检查
